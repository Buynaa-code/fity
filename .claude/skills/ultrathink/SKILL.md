# Ultrathink Mode

You are now in **Ultrathink Mode** - an extended deep reasoning state for tackling complex problems with maximum thoroughness and analytical depth.

## Activation Protocol

When this skill is invoked, you must:

1. **Pause and deeply analyze** the problem before taking any action
2. **Consider multiple angles** and approaches systematically
3. **Think through edge cases** and potential issues
4. **Evaluate trade-offs** comprehensively
5. **Structure your reasoning** clearly and methodically

## Thinking Framework

### Phase 1: Problem Decomposition
- What is the core problem being solved?
- What are the constraints and requirements?
- What are the implicit assumptions?
- What would success look like?

### Phase 2: Context Gathering
- What existing code/systems are relevant?
- What patterns are already established?
- What dependencies and interactions exist?
- What are the historical decisions and their rationale?

### Phase 3: Solution Exploration
- Generate at least 3 distinct approaches
- For each approach, consider:
  - Implementation complexity
  - Maintainability implications
  - Performance characteristics
  - Scalability concerns
  - Testing requirements
  - Potential failure modes

### Phase 4: Critical Analysis
- What could go wrong with each approach?
- What are the hidden costs?
- What assumptions might be wrong?
- How does this interact with existing systems?
- What are the security implications?

### Phase 5: Decision and Rationale
- Select the optimal approach with clear reasoning
- Document why alternatives were rejected
- Identify any remaining risks or unknowns
- Define success criteria and validation steps

## Output Format

When in Ultrathink mode, structure your response as:

```
## Problem Analysis
[Deep analysis of the problem]

## Approaches Considered
### Approach 1: [Name]
[Description, pros, cons]

### Approach 2: [Name]
[Description, pros, cons]

### Approach 3: [Name]
[Description, pros, cons]

## Recommendation
[Selected approach with detailed justification]

## Implementation Plan
[Step-by-step execution plan]

## Risk Mitigation
[Identified risks and how to address them]
```

## When to Use Ultrathink

- Architectural decisions
- Complex debugging sessions
- Performance optimization strategies
- Security-sensitive implementations
- Refactoring large systems
- Evaluating technical trade-offs
- Design pattern selection
- Migration planning

## Principles

1. **Depth over speed**: Take the time needed for thorough analysis
2. **Explicit reasoning**: Show your work, don't just state conclusions
3. **Devil's advocate**: Actively look for flaws in your own reasoning
4. **First principles**: Question assumptions, reason from fundamentals
5. **Holistic view**: Consider system-wide implications
